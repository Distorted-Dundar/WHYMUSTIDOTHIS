JavaScript Arrays, Objects & Array Iteration

  1. Objectives
  2. Demo
  3. Starting Code
  4. Stepped Solution
  5. Extensions for Independent Practice

Objectives

  • Students will write code directly in the JavaScript console and learn core data structures in JavaScript.
  • Students will learn how to store and reference information in arrays and objects to after inputting from the user and afterwards, outputting to the user
  • Students will learn to iterate through array objects incorporating for loops and conditional statements learned previously.
  • Students will use array methods to perform common tasks rather than writing them from scratch themselves.
  • Students will use arrays and objects to organize information in a format that is programmer friendly.

Key vocabulary: Array, Index, Methods, Objects, Keys & Values, Iteration

  const username = prompt("Enter a username");
  const password = prompt("Enter password");
  
  const login_cred = [
    { username: "redUser", password: "rose", color: "red" },
    { username: "greenUser", password: "grass", color: "green" },
    { username: "blueUser", password: "blueberry", color: "blue" },
  ];
  
  user_obj = login_cred.find((user) => user.username == username);
  
  if (user_obj) {
    if (user_obj.password == password) {
      console.log("Welcome " + user_obj.color + " user!");
    } else {
      console.log("Access Denied: Invalid password.");
    }
  } else {
    console.log("Access Denied: Invalid username.");
  }
  1. Run code and enter “redUser” for username and “rose” for password demonstrating to students how the code runs. Have the students try to guess login and password info so they get a better understanding of the code.
  2. After a couple of tries, show them “greenUser” with “grass” and “blueUser” with “blueberry”.
  3. Discuss whats happening with the students.
  4. Ask them “How do you think this could be set up behind the scenes?” Be sure to answer the following points.
    • How am I requesting a username? Password?
    • How would you set it up in the background? What would you use in your code? Explain.
  5. Talk about how we need more advanced ways to store and access information in order to remember information and successfully run code like this.
  6. Be sure to emphasize that something like this may be used in social media. However, not this exactly, as anyone would be able to inspect the page and see all the users’ usernames and passwords. That’d be terrible! We’re focusing on basic data structures today, not data security. Make sure students understand that they’d need much more than a simple if/else to protect user data.

Instructor Note: The concepts in this lesson are best explored through the practice problems. Try to keep the lesson as short as possible - that means you might only need to cover one or two examples of each concept. After that students are probably ready to start the tackling some problems. Keeping this intro short will give students more time to practice so they don’t finish this lesson in their panic zones.

Extensions for Independent Practice

Mild

Musician’s Bio (Object)

  • Create an object called “musician” with properties: name, instrument, age, and yrs_experience. Fill in these values yourself.
  • Print using console.log() a one-sentence biography about this musician using these properties.
  • Add the boolean property “prodigy” to the object which is true if yrs_experience is greater than 10 and if the age is less than 18.

Colors (Arrays)

  • Declare an array of 9 colors.
    • Print the color at index 4.
    • Add the color “chartreuse” to the array.
    • Print the index of the color “blue” in your array.

Medium

Student Info (Objects)

  • Create an object called “student” with properties: first name, last name, id, and grades (array). Fill in the property values as you wish. Give the student 4 grades (out of 100) as an array.
  • Write a program that accesses this object and does the following:
    • Print the full name (first name + last name),
    • Print the student’s gpa (average grade / 25),
    • Add a property to the object called “honors”. If the gpa is more than or equal to a 3.5, it should say true, otherwise, false.
