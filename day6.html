JavaScript Data Types, Conditionals & For Loops

Objectives

  • Students will write code directly in the JavaScript console and learn core datatypes in JavaScript.
  • Students will learn how to store information in variables and collect it from users.
  • Students will use conditionals to allow their code to respond to specific user inputs.
  • Students will use for loops to perform repetive tasks without repetitive code.

Key vocabulary: String, Boolean, Variable, Concatenation, Conditional, For Loop

  2. When you demo this code to student hide the code and only display the console so they just see the output.
  3. Run the code and ask a student to give you a random single digit number to type into the prompt.
  4. Run the code and examine the output with students.
  5. Rerun the program, and this time, enter a student’s name into the prompt instead of a single digit number.
  6. Ask students to predict what would happen before you hit enter and examine the output with students.
  7. Rerun the program with again, and this time enter a number greater than 9.
  8. Ask students to predict what would happen before you hit enter and examine the output with students.
  9. Ask students to make a list of all the things this program can do just from they’ve observed as output for all 3 examples.
  10. As students share out make sure they walk away with the following takeaways– This program can take in user input and:
    • check every number up to your secret number and return a specific output (OH! Your number is) when it reaches your secret number
    • check if the user entered a number or text, and return a different value for both
    • check if the user entered a number greater than 9, in which case the program will not run
  11. Tell students that putting together this program requires learning about JavaScript data types and learning really powerful programming tools, which will lay the foundation for the awesome things that they’ll build throughout this course.

The concepts in this lesson are best explored through the practice problems. Try to keep the lesson as short as possible - that means you might only need to cover one or two examples of each concept. After that students are probably ready to start the tackling some probelms. Keeping this intro short will give students more time to practice so they don’t finish this lesson in their panic zones.

  1. Ask students to predict how many times the word hello will appear in the console.

  2. Run the code and take time to breakdown the syntax of the for loop.

  3. Explain to students that a for loop takes three statements separated by a semicolon. In short, it’s easiest to think of these three statements as the start where…, while…, and go up by…. Reading these three statements for the above example, it would be “Start where i is 0, and keep going while i is less than 7, going up by 1 each time.”

  4. Ask students what needs to be changed in order to make the console.log statement run ten times.

Mild

Data Types & Conditionals

  • Create a new program which checks if a user is old enough to vote. If the user is 18 years or older they should receive a message that says You're old enough to vote! Otherwise they should receive You're not old enough to vote.

    • Include other age cutoffs for activities like driving, renting a car, or retiring.

  • Creat a program that tests to see if a number is positive or negative.

For Loops
