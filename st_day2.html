Objectives

  • Analyze and efficiently write structured HTML that implements CSS flexbox and grid layouts.
  • Use flexbox to format a webpage.
  • Describe sections of the website using CSS-driven vocabulary.

Key vocabulary: display: flex, flex-direction, justify-content, Flex Container, Flex Item, main axis, cross axis

What We’re Building

Today we will be formatting a pre-styled website using Flexbox. As we saw yesterday, CSS does a great job at styling our site, but we need an efficient way to organize different components into rows and columns with different lengths and heights and in different positions on the screen (e.g. centered, right-aligned, evenly spaced out, etc.)

  1. You’ve done it! Those are the basics of Flexbox. We activated flex to containers using display: flex;, we specified the direction in which items are organized (i.e. by row or column) using flex-direction, and we adjusted the positioning using justify-content.

  2. Now, we will look at a different layout tool in CSS called “Grid”. Don’t worry if you haven’t mastered all of flexbox yet - the cheat sheet is there for you as a reference.

  3. Now, we’re going to use CSS Grid to turn our list of items into this:

  .wide {

  • The grid-column: span 5; should elongate “Item 3” 5 units in width. Since the grid is compromised of 5 equally spaced columns, the item should take up the entire row.
  • The grid-row: span 2; should elongate “Item 6” 2 units in length. In other words, it should take up 2 rows, but remain 1 unit in length.
  1. Ask or tell students that if we wanted an item to be a large square taking up 3 units of width by 3 rows of length for example, we can code it in the following way. (Note: hover over the items to see how the color shows the size of the item.)

4


									
  .huge {
    grid-column: span 3;
    grid-row: span 3;
  }
  1. Notice that although we have the .huge div spanning 3 rows, it only appears to span 2. These weird things happen sometimes when we’ve given strict rules for how many rows an item should take up, but not how big each row should be. If you were to inspect, you’d see that one of the rows has collapsed to be 0px tall, which is a bizarre behavior. One solution is to specify row size with grid-template-rows, but that will actually still leave empty gaps in our grid. The alternative that we’re going to use is grid-auto-flow: dense; to indicate that empty spaces should be filled in with later items, even if the order is wrong. This feature should be put in the parent container. Think of it as a setting you want to adjust for the whole container. Code it in so it looks like the following, run it, and hover over the area to see the change in the display.

  .complexGallery {
    display: grid;
    grid-template-columns: repeat(5, 1fr);
    grid-auto-flow: dense;
  }

Stretch Goals

For all of today’s exercises, we used sections to create new activities, and then divs to represent both the parent and child elements of our various containers. The extensions today involve recreating the layout of popular or well-known sites online - If you want to recreate an entire layout in the limited time we have, you’ll need to emphasize just the format, not the content. Unless otherwise directed, please use placeholder words like “item” or “article” or “image” or “listing” or “tile” the same way we did in the activities.

If you want to rely on our default styles you can just keep adding your new work to the bottom or top of our existing HTML page, or you can create a totally new project if you’d prefer to start from scratch.

Mild

  • Add <a> links to your the simple nav so that clicking on a div actually takes you to a location. You can link out to sites like Google and Gmail as placeholders for now.
  • Replace all the “item” elements in the basic with something you can think of 12 of - birthstones, zodiac signs, angry men, perfect squares… the sky’s the limit!
  • Customize the content of each card to replicate the “plan” options from your favorite streaming service (e.g. “ad free!” v. “with ads”).

Medium

  • Recreate a Pinterest board. The easiest way to do this is to create a grid with 3-4 columns, and then make each column a flexbox container.
  • Recreate the airbnb search page - you’ll need to be clever about how to make the “map” element appear alongside the result tiles.

Spicy

  • Recreate any newspaper’s homepage. Any will do, but the New York Times will get extra tricky if you try to perfectly replicate the horizontal and vertical lines with the border-bottom and border-left and border-right properties.
