Objectives

You will learn to:

  • Use core datatypes in JavaScript and store information in variables and collect user input.
  • Use conditionals to respond to specific user inputs.
  • Utilize for loops to perform repetitive tasks without repetitive code.

Key vocabulary: String, Boolean, Variable, Concatenation, Conditional, For Loop

What We’re Building

We’re going to create dynamic programs that respond to user input. To do so we’ll need to learn about about JavaScript data types and several powerful programming tools, which will lay the foundation for the awesome things that we’ll build today and throughout this course.

  23. While it’s great to have a personal greeting for Beyonce, imagine how long it would take to write a greeting to several people at one time. Here’s an example of a for loop that would help us do that in 2 lines of code:

    1
2
3

    		
											
    for (let i = 0; i < 7; i++) {
  console.log("hello");
}

  24. A for loop takes three statements separated by a semicolon. In short, it’s easiest to think of these three statements as the start where…, while…, and go up by…. Reading these three statements for the above example, it would be “Start where i is 0, and keep going while i is less than 7, going up by 1 each time.”

  25. In another example, in order to make the for loop above run ten times, you would have to write i < 10 instead of 7 for the “while…” portion.

  26. Remember that the term i isn’t special or reserved, it’s just the most commonly used iterator. Each of the following options would also work:

    1
2

4

  console.log("hello");
}

for (let cheese = 0; cheese < 7; cheese++) {
  console.log("hello");
}

  27. We can also use the placeholder i to keep track of our count. If we wanted to count more visibly (instead of behind the scenes), we could do something like this:

    1
